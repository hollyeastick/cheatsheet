HTML Fundamentals

Accessibility

Tag Name Description Notes

<strong>

 		Strong Accessible tag allowing screen readers to emphasize strongly the content through a change in tone. Styled in italics. Different from the bold tag which doesn't trigger a change in tone for screen readers.

<em>

 		Emphasize Accessible tag allowing screen readers to emphasize the content through a change in tone. Styled in italics. Different from the italics tag which doesn't trigger a change in tone for screen readers.

Tables

Tag Name Description Notes

<table>

 		Table Table wrapper First element when creating a table

<thead>

 		Table Head Contains headings for the table columns Nest table row and table headings inside

<tbody>

 		Table Body Contains main content of table Used after table head

<th>

 		Table Heading Contains the title of table headings Nest inside Table Head or use alongside Scope attribute for a row heading

<tr>

 		Table Row A row containing Table Data

<td>

 		Table Data A cell containing data
